
# Nest js Authorization

Authenticte user login and signup and Enjoy Random joke.


## Features

 -  [Signup]
 -  [Login]
 - [View Profile]
 - [Read Random Joke]
 - [Logout]
## API Reference

#### Signup

```http
  Post /api/users/signup

```


#### Login

```http
  GET /api/users/Login
```
#### View Profile

```http
  GET /api/users/me
```
#### Random Joke

```http
  GET /api/random-joke
```
#### Logout

```http
  Post /api/users/logout

```
